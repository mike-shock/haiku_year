# Haiku Year (俳句の年)
This is a memorial application.
I'm gonna collect my haku in one place so that there will be a haiku for each day of the year.

The application in Go (cross-platform for GNU/linux, MS Windows, Android), written with Fyne,
can be run to read today's haiku.

I've got some photos and even short videos illustrating my haiku, but I'll not include 'em.
