# Haiku Year (俳句の年)
This is a memorial application.
I'm gonna collect my haku in one place so that there will be a haiku for each day of the year.

A GUI application written in Go (cross-platform for GNU/linux, MS Windows, Android), with the help of Fyne.
May be run every day to read today's haiku.

I've got some photos and even short videos illustrating my haiku, but I'll not include 'em.

I haven't decided yet: to include or not some early drafts, which don't comply with the rules of haiku poetry.
